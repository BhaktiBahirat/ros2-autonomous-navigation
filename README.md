
# ROS2 Autonomous Navigation 🚀

## 📌 Overview
This project demonstrates autonomous navigation using the ROS2 Navigation2 (Nav2) stack on:
- Custom robot in Gazebo
- TurtleBot3 simulation

The system performs waypoint-based navigation and reaches goals without using RViz.

---

## ⚙️ Features
- Custom robot simulation
- TurtleBot3 navigation
- Waypoint-based navigation
- Goal execution using Python node
- No RViz pose estimation dependency
- Pose estimation using TF frames

---

## 🛠️ Tech Stack
- ROS2 (Humble)
- Nav2 Stack
- Gazebo
- Python

---


## 💡 Key Highlight
This project enables fully autonomous navigation by sending goal poses programmatically instead of using RViz.
